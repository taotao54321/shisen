# Shisen-Sho

Simple Shisen-Sho game.

![Shisen-Sho screenshot](screenshot.png)


## Prerequisites

* [Meson](https://mesonbuild.com/)
* [SDL2](https://www.libsdl.org/)
* [fmt](https://fmt.dev/latest/)


## Build

```sh
$ mkdir build
$ meson setup build
$ ninja -C build
```


## Control

|             |        |
| -           | -      |
| left-click  | pick   |
| right-click | cancel |
| h           | hint   |
| q           | quit   |
| r           | reset  |
